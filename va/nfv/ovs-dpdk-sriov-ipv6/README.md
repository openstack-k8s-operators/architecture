# VA NFV OVS-DPDK-SRIOV with IPv6

If you are looking for information on how to deploy the OvS DPDK & SRIOV VA with IPV6, then
please see the
[Validated Architecture - NFV/OVS-DPDK-SRIOV with IPv6](../../../examples/va/nfv/ovs-dpdk-sriov-ipv6/README.md).

This directory, `architecture/va/nfv/ovs-dpdk-sriov-ipv6/`, exists so that the
[kustomization.yaml](../../../examples/va/nfv/ovs-dpdk-sriov-ipv6/kustomization.yaml)
in the examples directory for the OvS DPDK & SRIOV VA with IPV6, reference it by path as a
component. Its contents are likely uninteresting unless you want to
understand how kustomize was implemented in this repository.
