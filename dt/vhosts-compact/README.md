# DT: VHOSTS-COMPACT

If you are looking for information on how to deploy the VHOSTS-COMPACT DT, then
please see the
[VHOSTS-COMPACT README in the examples directory](../../examples/dt/vhosts-compact/README.md).

This directory, `architecture/dt/vhosts-compact/`, exists so that the
[kustomization.yaml](../../examples/dt/vhosts-compact/kustomization.yaml)
in the examples directory for the VHOSTS-COMPACT DT, reference it by path as a
component. Its contents are likely uninteresting unless you want to
understand how kustomize was implemented in this repository.
